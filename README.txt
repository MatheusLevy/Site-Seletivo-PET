Site sobre um evento fictício de hackthon.
O evento se chama "Hack the world".

Os requisitos do site são:
*Conter:
    1. Header
    2. Slider
    3. Seleção sobre:
        3.1. Localização
        3.2. Data de início
        3.3  Sobre o evento
    4. Seção Cronograma
    5. Seção submissão de trabalhos
    6. Formulário de inscrição
    7. Contato
    8. FAQ  
    9. Footer

Deve ser feito em HTML, CSS e JavaScript.
